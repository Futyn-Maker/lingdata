# Lemmangrams

Скрипт `lemmatized.py` лемматизирует все слова текста файла, который в нём указан.

Скрипт `ngrams.py` создаёт частотный список N-грамм, работая с леммами (файл, полученный при выполнении предыдущего скрипта).

## Зависимости

Для работы скрипта необходима библиотека `pymystem3` для лемматизации, а также `nltk` — для составления списка N-грамм. Всё можно установить, выполнив команду:
```
pip install -r requirements.txt
```
